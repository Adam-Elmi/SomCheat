       # Rules for Variable Names
       Marka la magacaabayo variable-ka waxa jira sharciyo u gaar ah oo ku saabsan sida lo magacaabo variables-ka. Shuruucdaas waxa ka mid ah:
       #### <Colorize color="somcheat-blue" is_both={true}>1. Magaca variable-ku kuma bilaabmi karo tiro </Colorize>
       <Code>
        ```js
        let 1user = "Adam"; // Error
        ```
       </Code>
       <Message type="error">
    	**SyntaxError:** Invalid or unexpected token
       </Message>
       <Message type="note">
       	Laakiin magaca waxaad ku bilaabi karta **$** ama **_** ama **xaraf**. Tusaale:
          <Code>
            ```js
		let $price = 10;
		let _discount = 0.02;
		let total =  9.8;
            ```
          </Code>
          Inta wixii ahayn magaca kuma bilaabi kartid.
       </Message>
       #### <Colorize color="somcheat-blue" is_both={true}>2. Ma isticmaali kartid ereyo bixinta u gaar ah luuqada</Colorize>
       Marka magacaabaysid ha isticmaalin ereyo bixinta u gaarka ah luuqada (**reserved keywords**). Tusaale:
        <Code> 
        ```js
	let let = 5; // Error
        ```
        </Code>
        <Message type="error">
        **SyntaxError:** **let** is disallowed as a lexically bound name 
       </Message>
      #### <Colorize color="somcheat-blue" is_both={true}>3. Case-sensitive</Colorize>
     Haddii ay jiraan laba variables/doorsame oo isku magac ah laakiin ku kala duwan xarfaha waaweyn iyo kuwa yaryar waxa lo aqoonsanaya inay yihiin laba variables oo kale duwan. Tusaale:
     <Code>
     	```js
     	// Example 1
	let name = "Adam";
	let Name = "Elmi";

	console.log(name); // Adam
	console.log(Name); // Elmi
     	```
     </Code>
     <Code>
    	```js
	// Example 2
	let age = 25;
	console.log(Age); // Error
    	```
     </Code>
    <Message type="error">
     **ReferenceError:** **Age** is not defined
    </Message> 
    ### Hababka kala duwan ee lo qori magaca variable-ka
    ##### <Colorize color="somcheat-blue" is_both={true}>Camel Case</Colorize>
    Habkan waxa weeye in kalmada ugu horeysa laga wada dhigo xuruuf yaryar, kalmadaha kalena xarafkooda ugu horeya waxa laga dhigaya xaraf weyn, xuruufta kalena xuruuf yaryar. Tusaale:
    <Code>
      ```js
	let firstName = "Adam";
	let lastTimeVisited = "Yesterday";
      ```
    </Code>
    ##### <Colorize color="somcheat-blue" is_both={true}>Pascal Case</Colorize>
    Habkan waxa weeye in kalmad kasta xarafkeeda kowaad laga dhigo xaraf weyn, inta kalena xuruuf yaryar. Tusaale:
    <Code>
      ```js
	let UserId = 101;
      ```
    </Code>
    ##### <Colorize color="somcheat-blue" is_both={true}>Snake Case</Colorize>
    Habkan waxa weeye in dhamaan kalmadaha laga wada dhigo xuruuf yaryar sido kalena kalmadahana loo kala dhexeesiyo **_**(**underscore**). Tusaale:
    <Code>
      ```js
	let user_name = "Adam";
      ```
    </Code>

        # Data Types
        Xogta variable-ka lagu kaydinayo waxa lo kala qaybiya laba nooc oo kale ah 
        <Colorize color="somcheat-blue" is_both={true}>Primitive Data Types</Colorize> iyo
        <Colorize color="somcheat-blue" is_both={true}>Reference Data Types(Non-Primitive)</Colorize>.
        #### <Colorize color="somcheat-blue" is_both={true}>1. Primitive Data Types</Colorize>
        Waa xog hal qiimo ka kooban oo la bedeli karin.

        <Colorize color="somcheat-gray" is_both={true}>Noocyada xogta Primitive data ah waxa ka mid ah: </Colorize>
        #### <Colorize color="somcheat-blue" is_both={true}>a. String</Colorize>
        Waa xogta qoraalka ah. Xogtaas waxa la dhex galiya " " ama ' ' ama \` ` si loogu aqoonsado inay yihiin <Colorize color="somcheat-blue" is_both={true}>string</Colorize>.
        <Code>
            ```js
            let firstName = "Adam";
            let middleName = 'Eid';
            let lastName = `Elmi`;

            ```
        </Code>

        #### <Colorize color="somcheat-blue" is_both={true}>b. Number</Colorize>
       	Waa xogta tirooyinka ah. Waa dhamaan xogta tirooyinka ah: **integers** (tirooyinka), **floats** (tirooyinka ah jajab tobanle) iyo xogta gaar ah sida **'NaN'**, **'Infinity'** kuwaas oo iyana lo aqoonsado inay yihiin xog tirooyin ah.
        <Code>
            ```js
            let items = 15; // integers
            let price = 10.5; // floats 
            ```
        </Code>
 	
	#### <Colorize color="somcheat-blue" is_both={true}>c. Boolean</Colorize>
       	<Colorize color="somcheat-blue" is_both={true}>Boolean</Colorize> waa nooc ka mid ah xogta oo kaliya yeelan karta laba qiimood: run (true) ama been (false).
        <Code>
            ```js
            let isOnline = true;
            let is_student = false; 
            ```
        </Code>

	#### <Colorize color="somcheat-blue" is_both={true}>d. Undefined</Colorize>
   	<Colorize color="somcheat-blue" is_both={true}>Undefined</Colorize> waa nooc ka mid ah xogta u variable-ka u qaadanayo haddii la qeexin qiimaha <Colorize color="somcheat-blue" is_both={true}>Variable-ka</Colorize> lagu kaydinayo taas oo micnaheedu u yahay qiimihii ayaa maqan.
        <Code>
            ```js
            let location; // undefined
            ```
        </Code>

	#### <Colorize color="somcheat-blue" is_both={true}>e. Null</Colorize>
       	<Colorize color="somcheat-blue" is_both={true}>Null</Colorize> waa nooc ka mid ah xogta taas oo micnaheedu u yahay **"qiimo madhan"** oo aan waxba u taagneyn. 
        <Code>
            ```js
            let file = Null;
            ```
        </Code>
