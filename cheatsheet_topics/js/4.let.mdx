    # Let
     <Colorize color="somcheat-blue" is_both={true}>Let</Colorize> waa erey bixin lo adeegsado in variable(doorsame) lagu qeexo.
     <Steps title="Astaamaha uu iska leeyahay - Let"  data={[

        <Colorize color="somcheat-gray" is_both={true}>Tusaale: </Colorize>
        #### <Colorize color="somcheat-blue" is_both={true}>1. Reassignment</Colorize>
        <Code>
            ```js
            // Reassignment
            let price = 10;
            price = 20;
            console.log(price); // natiijada: 20
            ```
        </Code>
        #### <Colorize color="somcheat-blue" is_both={true}>2. Redeclaration</Colorize>
        <Code>
            ```js
            // Redeclaration
            let color = "red";
            let color = "blue";
            console.log(color); // natiijada: Error
            ```
        </Code>
        <Message type="error">
        **SyntaxError:** Identifier ***'color'*** has already been declared
        </Message>
        #### <Colorize color="somcheat-blue" is_both={true}>3. Block scope</Colorize>
        <Code>
            ```js
            // Example 1: Block scope
            function add(a, b) {
              let result = a + b;
              console.log(result); // natiijada: 5
            }
            add(2, 3);
            console.log(result); // Error
            ```
        </Code>
        <Message type="error">
        **ReferenceError:** ***result*** is not defined
        </Message>
        <Code>
            ```js
            // Example 2: Block scope
            if(true) {
              let user = "Admin"; 
              console.log(user); // natiijada: Admin
            }
            console.log(user); // Error
            ```
        </Code>
        <Message type="error">
        **ReferenceError:** ***user*** is not defined
        </Message>
        <Message type="note">
        **Variables**-ka lagu dhex qeexo **'\{}\'**, kaliya waxa laga heli kara meesha lagu dhex qeexay.
        </Message>
        #### <Colorize color="somcheat-blue" is_both={true}>4. Hoisting</Colorize>
        <Code>
            ```js
            // Hoisting
            id;
            let id = 101;
            console.log(id); // natiijada: Error
            ```
        </Code>
        <Message type="error">
        **ReferenceError:** Cannot access ***'id'*** before initialization
        </Message>
        #### <Colorize color="somcheat-blue" is_both={true}>5. Undefined Value</Colorize>
        <Code>
            ```js
            // Undefined Value
            let name;
            console.log(name); // natiijada: undefined (lama qeexin qiimaha la kaydinayo)
            ```
        </Code>
